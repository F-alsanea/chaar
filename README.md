# شعار العقارية

موقع شركة شعار العقارية - منصة عقارية متكاملة لعرض العقارات وإدارة طلبات التملك.

## المميزات
- عرض العقارات المتاحة للبيع والإيجار
- نموذج طلب تملك عقار
- لوحة تحكم لإدارة العقارات والطلبات
- حماية شاملة (CSRF, CSP, Rate Limiting, bcrypt)

## التقنيات
- React + TypeScript + Vite
- TailwindCSS + Motion (Framer Motion)
- Supabase (PostgreSQL)
- Vercel Serverless Functions

## التشغيل المحلي
```bash
npm install
npm run dev
```

## Environment Variables
انسخ `.env.example` إلى `.env.local` واملأ القيم المطلوبة.
